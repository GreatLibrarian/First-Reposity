High Scores Home Guessing Game

Hello!

Welcome to the Home Page of my application!

Clicking "High Scores" will direct you to a High Scores Page where you can create new High Scores for the table, and clicking "Guessing Game" will direct you to a game where you guess the right number and try to get ten correct guesses within 3 lives!

Originally, I made all three pages linked to each other in some way (ie. Having the home page make the settings for the Guessing Game), but quickly realized that transferring data between different pages with javascript without a database was unfeasible. To replace that, enjoy a dropdown with some silly videos!
